# FP&A Analytics Portfolio
### End-to-end financial planning & analysis simulation | Antonio Heredia

A full-stack FP&A project built on a simulated SaaS company dataset. 
Designed to demonstrate the analytical and technical skills required 
in modern FP&A roles — from raw data to executive reporting.

## What this project covers
- Revenue forecasting and variance analysis (budget vs actual)
- Cohort retention modeling
- Operational expense breakdown
- Executive dashboard for management reporting

## Tech stack
| Tool | Purpose |
|------|---------|
| SQL / SQLite | Database design and analytical queries |
| Python / Pandas | Forecasting model and cohort analysis |
| Jupyter Notebooks | Documented analysis and logic walkthrough |
| Power BI | Executive dashboard |
| HTML / CSS | Public portfolio page |
| Claude (AI) | Used openly throughout — documented in repo |
